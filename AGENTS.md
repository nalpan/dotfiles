# 計画の作り方

原則は以下のようにしてください、別途指示があればその指示を優先してください。

- 計画は作業ディレクトリの./local-plansフォルダにMarkdown形式で保存してください。
- 計画のフォーマットは以下の通りにしてください。

```markdown
# 計画のタイトル

計画の概要を簡潔に記載してください。

## 目的

計画の目的を明確に記載してください。

## 実行計画

### Phase 1: フェーズ1のタイトル

- 目的
- 完了条件

#### Step 1: ステップ1のタイトル

- [ ] タスク1
- [ ] タスク2
- [ ] タスク3

#### Step 2: ステップ2のタイトル

- [ ] タスク1
- [ ] タスク2
- [ ] タスク3

以後同様にPhaseとStepを追加してください。

```
